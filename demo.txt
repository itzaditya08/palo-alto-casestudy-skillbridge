1. Start the Backend (Terminal 1):
cd backend
source venv/bin/activate
uvicorn main:app --reload

2. Start the Frontend (Terminal 2):
cd frontend
npm run dev

3. To show the automated tests passing:
(Open a new terminal tab in the frontend folder)
npm run test

4. TO TRIGGER THE FALLBACK (The "Senior Flex"):
GOOGLE_API_KEY="broken_key_123" uvicorn main:app --reload
